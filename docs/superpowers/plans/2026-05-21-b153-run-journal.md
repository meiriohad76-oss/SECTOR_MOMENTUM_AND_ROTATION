# B-153 Run Journal Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add the append-only run-journal foundation and wire dashboard scoring runs into it for later recommendation debriefing.

**Architecture:** Create `src/run_journal.py` as a pure SQLite boundary. Keep dashboard wiring out of B-153.1 so the schema and persistence contract are tested before Streamlit writes production run evidence. B-153.2 keeps conversion and append safety in the same pure module, then calls it from `app.py` after BLUF decisions are built.

**Tech Stack:** Python stdlib `sqlite3`, `dataclasses`, `json`, `pathlib`, pytest.
